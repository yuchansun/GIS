## 開發環境

- XAMPP（Apache + MySQL + PHP）
- 專案路徑：`xamppfiles/htdocs/GIS`

## 本地啟動方式

1. 安裝並啟動 [XAMPP](https://www.apachefriends.org/)。
2. 將本專案放在 `htdocs/GIS` 資料夾下。
3. 啟動 Apache（如有用到資料庫則一併啟動 MySQL）。
4. 瀏覽器開啟 `http://localhost/GIS`。

## 協作流程
